## Instructions:

### Requirements:

[Deno](https://deno.land/)

```bash
curl -fsSL https://deno.land/x/install/install.sh | sh
```

[Lume](https://lumeland.github.io/) (A SSG for Deno):

```bash
deno install --unstable -A https://deno.land/x/lume/cli.js
```

### Usage

Start a local server:

```bash
lume -s
```
